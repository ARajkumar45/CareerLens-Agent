# 🤖 CareerLens AI Agent

AI-powered career coach agent that helps you 
find jobs, research skills and track AI trends.

## Features
- 🔍 Job search across 13 roles + 8 locations
- 📚 Skills research for any tech role
- 📈 Latest AI industry trends
- 💬 Conversation memory — remembers context
- 🎯 Powered by LangGraph agent orchestration

## Tech Stack
- LangChain + LangGraph — agent orchestration
- NVIDIA LLM — llama-3.3-70b-instruct
- Tavily — real-time web search
- Streamlit — chat UI
- InMemorySaver — conversation memory

## Setup
1. Clone the repo
2. Copy example.env to .env
3. Add your API keys
4. Install dependencies:
pip install -r requirements.txt
5. Run:
streamlit run app.py
